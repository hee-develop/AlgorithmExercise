## 2xn 타일링

문제 링크 : https://www.acmicpc.net/problem/11726

- `2 x 1`일 때 1가지 방법(`|`)
- `2 x 2`일 때 2가지 방법(`||`, `=`)
- `2 x 3`일 때 3가지 방법(`|||`, `|=`, `=|`)
- `2 x 4`일 때 5가지 방법(`||||`, `||=`, `|=|`, `=||`, `==`)
- `2 x 5`일 때 8가지 방법(생략)

이전 값의 끝에 `|`나 `=`을 붙여 경우의 수를 늘리고 있다.
예를 들어, n=3일 때,
이전 값 n=2의 결과 `||`에 `|`을 붙여 `|||`을 만들고,
`=`에 `|`을 붙여 `=|`을 만들고,
n=1의 결과 `|`에 `=`을 붙여 `|=`을 만들었다.

중요한 건 이전 두 개의 값만 활용한다는 건데,
이는 타일의 크기가 2라서 그렇다.

```cpp
#include <stdio.h>

int main()
{
    int n;
    scanf("%d", &n);

    int arr[n+1];
    arr[1] = 1;
    arr[2] = 2;

    for (int i = 3; i <= n; i++)
    {
        arr[i] = arr[i-1] + arr[i-2];
    }

    printf("%d\n", (arr[n]) % 10007);

    return 0;
}
```
